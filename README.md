# Application de Prediction de Prix Immobilier

## Description
Application web interactive pour predire les prix immobiliers
basee sur l'apprentissage supervise.

## Acces
[Lien vers l'app]((https://house-price-app-9nozdkxkhnaxz2pfzz6aif.streamlit.app/))

## Fonctionnalites
- Exploration interactive des donnees
- Prediction de prix avec Gradient Boosting
- Analyse des biais par quartier

## Installation locale
pip install -r requirements.txt
streamlit run app.py

## Technologies
- Python 3.10
- Streamlit
- Scikit-learn
- Pandas
- Matplotlib
